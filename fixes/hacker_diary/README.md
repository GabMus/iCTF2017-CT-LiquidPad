# How to apply this patch

Move views.patch to `/opt/ctf/hacker_diary/ro/diary/`

```
mv views.patch /opt/ctf/hacker_diary/ro/diary
```

Change directory to the diary folder

```
cd /opt/ctf/hacker_diary/ro/diary

```

Patch the service

```
patch < views.patch
```
